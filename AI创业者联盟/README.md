> "连接AI创业者，让价值自由流动"

---

## 关于我们

AI创业者联盟（AI Entrepreneurs Alliance，简称AIEA）是由AI领域创业者、投资人和专家共同发起的非营利性组织，致力于成为AI时代的价值创造中枢。

我们借鉴华尔街的运作模式，构建一个集资源整合、价值发现、信用背书于一体的AI创业生态圈。

## 会员等级

| 等级 | 称号 | 核心权益 |
|------|------|----------|
| Lv.5 | 创始成员 | 理事会席位，最高决策权 |
| Lv.4 | 核心成员 | 深度参与，优质项目优先跟投 |
| Lv.3 | 活跃成员 | 项目参与，资源对接 |
| Lv.2 | 合作伙伴 | 资源提供，投资对接 |
| Lv.1 | 观察员 | 体验试用，公开资源 |

## 孵化流程

```
创意 → 评审 → 孵化 → 加速 → 融资 → 退出
```

联盟提供从0到1的全方位孵化支持，包括资金、算力、导师、资源对接等服务。
