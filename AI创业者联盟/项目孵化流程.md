## 阶段一：创意阶段 (Idea Stage)

### 阶段目标
- 验证想法的可行性
- 组建初始团队
- 完成市场调研

### 入口条件
- 项目发起人必须是联盟成员
- 提交项目意向书
- 找到至少1名联合创始人

### 阶段产出
| 产出物 | 说明 |
|--------|------|
| 项目意向书 | 描述项目愿景、解决的问题 |
| 市场调研报告 | 市场规模、竞品分析 |
| 初始团队 | 至少2名核心成员 |
| MVP方案 | 产品最小可行方案 |

### 阶段资源
- **导师对接**：1-2次方向指导
- **知识库**：竞品分析模板
- **团队匹配**：协助寻找联合创始人

### 阶段时长
- **标准周期**：4-8周
- **最长周期**：12周（可申请延期）

### 阶段评估
- 问题是否真实存在
- 团队是否具备执行力
- 市场规模是否足够

## 阶段二：评审阶段 (Review Stage)

### 阶段目标
- 正式立项
- 获取孵化资格
- 确定资源需求

### 评审流程
```
提交申请 → 形式审查 → 专业评审 → 投资委员会 → 立项决议
```

### 评审维度

| 维度 | 权重 | 评审要点 |
|------|------|----------|
| 团队 | 30% | 背景、互补性、执行力 |
| 市场 | 25% | 规模、增速、切入点 |
| 产品 | 20% | 创新性、可行性、壁垒 |
| 商业模式 | 15% | 盈利模式、变现路径 |
| 联盟契合 | 10% | 与联盟资源的协同 |

### 评审结果
| 结果 | 说明 |
|------|------|
| **通过** | 获得孵化资格，分配资源 |
| **有条件通过** | 需要补充某些条件后孵化 |
| **待定** | 观察期，3个月后再评审 |
| **不通过** | 可重新优化后再次申请 |

### 评审材料清单
- [ ] 项目计划书（15-20页）
- [ ] 产品Demo或原型
- [ ] 团队介绍
- [ ] 财务预测（3年）
- [ ] 资源需求清单
- [ ] 风险评估报告

## 阶段三：孵化阶段 (Incubation Stage)

### 阶段目标
- 完成产品开发
- 验证商业模式
- 积累种子用户

### 孵化服务

#### 3.1 资金支持
| 资源 | 额度 | 形式 |
|------|------|------|
| 种子资金 | 10-50万 | 可转债 |
| 算力资源 | 按需申请 | 配额制 |
| 云服务 | 5-20万额度 | 代金券 |

#### 3.2 导师支持
- **技术导师**：AI技术指导
- **商业导师**：商业模式、运营策略
- **战略导师**：行业洞察、资本运作

#### 3.3 资源对接
- 潜在客户对接
- 合作伙伴引荐
- 人才招聘支持

#### 3.4 基础设施
- 办公空间（可选）
- 法律合规支持
- 财务记账服务

### 阶段里程碑

| 里程碑 | 时间节点 | 完成标准 |
|--------|----------|----------|
| M1 | 第1个月 | 团队到位，产品规划完成 |
| M2 | 第3个月 | MVP完成，开始内测 |
| M3 | 第6个月 | 获取首批付费用户 |
| M4 | 第9个月 | 月活/收入达到指标 |
| M5 | 第12个月 | 完成A轮前准备 |

### 阶段考核
- **月度汇报**：进度更新、问题反馈
- **季度评估**：里程碑完成情况
- **中期评审**：第6个月进行全面评估

### 阶段成果
- 可演示的产品
- 验证的商业模式
- 种子用户数据
- 清晰的融资计划

## 阶段四：加速阶段 (Acceleration Stage)

### 阶段目标
- 快速扩大规模
- 完成A轮融资
- 建立竞争壁垒

### 加速服务

#### 4.1 融资加速
- 对接顶级投资机构
- 商业计划书打磨
- 路演训练
- 条款谈判支持

#### 4.2 资源加速
- 战略合作伙伴引荐
- 大客户对接
- 核心人才猎头

#### 4.3 认知加速
- 创始人培训课程
- 管理团队搭建
- 组织架构优化

### 加速周期
- **标准周期**：3-6个月
- **里程碑**：完成A轮融资

## 阶段五：退出阶段 (Exit Stage)

### 退出路径

| 退出方式 | 说明 | 适用场景 |
|----------|------|----------|
| 融资退出 | 创始团队回购/转让部分股份 | 团队想保持独立 |
| 并购退出 | 被大公司收购 | 行业整合时机 |
| IPO退出 | 独立上市 | 市场规模足够大 |
| 项目终止 | 清算重组 | 无法继续发展 |

### 退出流程
```
退出评估 → 退出方案制定 → 联盟收益计算 → 执行退出 → 收益分配
```

### 联盟收益分配

| 阶段 | 联盟持股 | 说明 |
|------|----------|------|
| 种子轮 | 5-10% | 孵化服务对价 |
| A轮后 | 3-5% | 加速服务对价 |
| 退出时 | 按持股比例 | 最终收益分配 |

- 里程碑追踪

### 会议制度

- 双周会：业务评审

## 常见问题

### Q: 项目失败怎么办？
A: 正常创业失败不追责，但会分析失败原因形成案例。如发现违规或严重失职，将影响成员信用评级。

### Q: 创始人中途退出怎么处理？
A: 根据贡献度和退出原因协商股权处理，保护项目和其他成员利益。

### Q: 项目方向需要调整怎么办？
A: 可申请Pivot，需重新评审并可能调整资源支持。
